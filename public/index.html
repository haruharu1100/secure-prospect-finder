Web制作 見込み客ファインダー

地域・業種を入れて、サイト未整備の店舗や小規模事業者をリスト化します。営業前の「探す、優先順位をつける、記録する、CSVにする」までを一画面で回せます。

発掘条件

Google Places APIキーがある場合は自動取得できます。ない場合も、検索リンク生成とCRM管理はそのまま使えます。

APIキーは保存しません。Google Cloudで Places API (New) を有効化したキーを使います。

公開後に入力すると、DM内のサンプルURLがHTTPSになります。ローカル利用中は空でOKです。

実行した業種は「済」と表示されます。どれを回したか迷わないための記録です。

営業テンプレ

候補ごとの「DM作成」から、無料診断DM・LP提案・HP提案を作れます。

このサイトの使い方
  1. 地域を入れて、営業したい業種にチェックします。
  2. APIキーを貼り、APIで候補取得を押します。キーなしなら検索リンク生成で手動発掘します。
  3. スコアが高い順に、サイトなし・SNS中心・口コミありの候補を見ます。
  4. DM作成で無料診断DMを送り、反応が来たらLP提案またはHP提案を送ります。
  5. 送ったらステータスを診断送付、ラフ提案、商談化に変えます。

契約率を上げる流れ: 無料診断DM → 業種別サンプルURL → 改善ポイント3つ → 1枚ラフ提示 → 10万円前後の制作提案 → Googleマップ改善の月額提案。

Googleマップ改善サポート: 口コミ購入ではなく、プロフィール整備、写真追加、投稿更新、口コミ返信、自然な口コミ依頼導線を商品化します。
